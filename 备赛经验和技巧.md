**输入输出优化**
```cpp
// 关闭同步, 取消cin和cout的绑定, 可以加速输入输出.
ios::sync_with_stdio(false);
cin.tie(0);
// 注意, 不能同时使用scanf和cin, 不能同时使用printf和cout, 否则会出问题.
// 即, 只能使用一种输入输出方式, 不能混用. 例如, 只能使用cin和cout, 不能使用scanf和printf.
```

**头文件引入**
- `#include <bits/stdc++.h>`: 万能头文件, 包括下面列出的所有.
- `#include <climits>`: 提供`INT_MAX`、`INT_MIN`、`LLONG_MAX`、`LLONG_MIN`等常量.
- `#include <algorithm>`: 提供`sort`函数等算法.

**开启long long类型的几种写法**
- `#define int long long`
- `typedef long long ll`
- `using ll = long long`

**常量定义**
- `const int MAXN = 1e5+5;` // 常用的数组大小上限, 根据题目要求调整.
- `const int INF = 1e9;` // int类型的一个较大的常量, 代表无穷大.
- `const ll LLINF = 1e18;` // long long类型的一个较大的常量, 代表无穷大.
- `const int MOD = 1e9 + 7;` // 常用的模数, 用于取模运算.


**设置最值(注意头文件引入)**
- 使用`INT_MAX`和`INT_MIN`来设置int类型的最值.
- 使用`LLONG_MAX`和`LLONG_MIN`来设置long long类型的最值.


**优化的几个思路**
- 凭借数学直觉.
- 对公式的化简.
- 打印数据处理的过程数据, 尝试发现规律.
