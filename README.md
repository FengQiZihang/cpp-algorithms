# cpp-algorithms

一个用于记录 C++ 算法题练习、题解思路和备赛笔记的仓库，当前主要包含蓝桥杯第十六届 B 组相关题目代码。

## 项目内容

- `第十六届B组/`
  - 按题目整理的 C++ 代码
  - `05画展布置/` 下包含多个版本的解法与配套数学思路说明
- `备赛经验和技巧.md`
  - 备赛过程中整理的常用技巧、输入输出优化、调试经验等
- `helloworld.cpp`
  - 本地环境初始化时的示例文件，可忽略其算法意义
- `编译文件夹/`
  - 本地编译产生的可执行文件目录，不纳入版本管理

## 当前已包含的题目

- 03 可分解的正整数
- 04 产值调整
- 05 画展布置
- 08 装修报价（当前为占位代码）

## 代码风格

仓库中的代码以竞赛/刷题风格为主，特点包括：

- 优先追求题目通过与实现简洁
- 常见写法如 `ios::sync_with_stdio(false);`、`cin.tie(0);`
- 在需要时使用 `long long` 避免溢出
- 同一题可能保留多个版本，便于比较不同思路

## 运行方式

### 使用 g++ 编译

```bash
g++ -std=c++17 -O2 -o main "第十六届B组/03可分解的正整数.cpp"
./main
```

如需运行其他题目，将文件路径替换为对应的 `.cpp` 文件即可。
